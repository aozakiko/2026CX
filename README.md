# 创新项目实践

本仓库整理了 STM32/Keil 实验工程。当前工作区中只纳入以“实验”结尾的实验目录，压缩包、驱动包、图片缓存和 Keil 编译输出已通过 `.gitignore` 排除。

实验目录已按照各 Keil 工程最后打开/保存痕迹排序编号，报告见 [实验报告.md](实验报告.md)。

各实验的主 Keil 工程统一放在内部 `Project/` 目录中。`04-内部温度传感器显示实验` 额外保留 `Reference-基础温度采集实验/`，用于存放旧版基础温度采集例程。

## 实验目录

| 编号 | 实验目录 | 主要内容 |
| --- | --- | --- |
| 01 | `01-指针式电子时钟实验` | RTC 获取时间，TFTLCD 绘制指针式时钟与数字时间 |
| 02 | `02-蜂鸣器控制实验` | GPIO 控制蜂鸣器与 LED 状态翻转 |
| 03 | `03-ESP8266-MQTT联网调试实验` | ESP8266 STA 联网、MQTT 连接与订阅控制 |
| 04 | `04-内部温度传感器显示实验` | STM32 内部温度 ADC 采样，串口与 TFTLCD 显示 |
| 05 | `05-FSMC-TFTLCD显示实验` | FSMC 驱动 TFTLCD，文本、色块和图片显示 |
| 06 | `06-光敏传感器采集实验` | 光敏传感器 ADC 采集与串口输出 |
| 07 | `07-USB虚拟串口通信实验` | USB CDC 虚拟串口收发与 LCD 状态显示 |
| 08 | `08-温湿度光照环境监测实验` | 内部温度、DHT11 湿度、光照采集，串口帧输出，MQTT 发布与手机前端显示 |
| 09 | `09-ESP8266温度MQTT发布实验` | ESP8266 MQTT 控制 LED 并发布内部温度 |
| 10 | `10-串口摄像头TFTLCD显示实验` | PC 摄像头串口传帧，STM32 校验并在 TFTLCD 显示 |
